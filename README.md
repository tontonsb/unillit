# Unillit

Feel illiterate in a foreign script? Un-illiterate yourself!

## Development

Prerequisites: node & git.

Clone the repo and install the dependencies:

```sh
git clone https://github.com/tontonsb/unillit.git
npm install
```

Launch dev server:

```sh
npm run dev
```

Type-Check, Compile and test if it builds:

```sh
npm run build
```

Check your code style and stuff:

```sh
npm run lint:eslint
```
